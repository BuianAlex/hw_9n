# Admin page + API

## The app created for learnig Node.js and React

## What you can

- Create new user
- Edit user
- ADD photo for user
- Delete users from table
- Import users from csv

## Config

```
    Node: 13.5.0
    NPM: 6.13.7
    MongoDB 3.6.3
```

## Instalation

    /client
      npm install
    /server
      npm install

## Start

    /client
      npm start
    /server
      npm start

    create first user
      /server/scripts
        node starterDB.js
